### 2.在windows环境

指定python解释器路径，下面代码需要放在脚边第1行:tv:

```
#!C:\Users\heliang\AppData\Local\Programs\Python\Python38-32\python.exe
```

### 3.如果是在Linux环境

则第1行 应该像下面这么写:iphone:

```
#!/usr/bin/env python
```
