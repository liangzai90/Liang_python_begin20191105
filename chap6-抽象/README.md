新函数

map(func, seq[, seq, ...])  对序列中的所有元素执行函数

filter(func, seq)  返回一个列表，其中包含对其执行函数时结果为真的所有元素

reduce(func, seq[, initial])  等价于func(func(seq[0], seq[1], seq[2]), ...)

sum(seq)  返回seq中所有元素的和

apply(func[, args[, kwargs]])  调用函数（还提供要传递给函数的参数）
